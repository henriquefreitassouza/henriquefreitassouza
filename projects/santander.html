Santander Mobile Redesign

O que é: uma sujestão de redesign do processo de pagamento de boletos para o aplicativo mobile do banco Santander.

Este trabalho é resultado de semanas de estudo usando o aplicativo Santander e observando pessoas que também o usam. O objetivo do projeto é simplificar para o usuário o processo de pagamento de uma nova conta pelo smartphone. O redesign é uma iniciativa própria, não tendo relação com o banco Santander. As principais mudanças foram a remoção do cartão de segurança, seguindo a linha dos bancos digitais, e a simplificação do formulário de preenchimento dos dados do boleto. A interface foi redesenhada com o objetivo de distribuir a informação de forma a evidenciar etapas de um processo. O visual segue uma estética alinhada aos padrões de interface da Apple e do Google, que contém elementos de navegação e interação já conhecidos pelos usuários.
