Santander Mobile Redesign

O que é: uma sujestão de redesign do processo de pagamento de boletos para o aplicativo mobile do banco Santander.

Este trabalho é resultado de semanas de estudo usando o aplicativo Santander e observando pessoas que também o usam. O objetivo do projeto é simplificar para o usuário o processo de pagamento de uma nova conta pelo smartphone. O redesign é uma iniciativa própria, não tendo relação com o banco Santander. As principais mudanças foram a remoção do cartão de segurança, seguindo a linha dos bancos digitais, e a simplificação do preenchimento dos dados do boleto. A interface foi redesenhada com os objetivos de distribuir a informação de maneira que faça mais sentido para os usuários e diminuir o número de passos para realizar a transação.
