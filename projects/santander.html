Santander Mobile

What is it: A complete redesign of the payment flow of the Santander internet banking mobile application.

This work is the result of weeks using the Santander mobile application and watching the people that use it as well. The goal of the redesign is to simplify the payment flow using the smartphone. The main changes were the removal of the security card, following the digital banks, and the simplification of data collection at every step that it is required. The user interface was redesigned to evidence that the user is walking through a process. The visual elements follow the interface guidelides defined by Apple and Google at their specifications and contain elements that are already familiar for most users.

Achievments:
  • It is easier and more intuitive to make payments;
  • Less cognitive efforts for the users by having known elements of their operational systems;
